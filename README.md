## Introduction

Aether Manager is a native Android application for rooted devices, supporting Android 8.0 (API 26) and above.<br>
Some highlight features:

- **Performance Profiles**: Switch between Balance, Performance, Gaming, and Battery Saver profiles
- **Kernel Tweaks**: 17+ individual tweaks across CPU, GPU, memory, I/O, network, and battery
- **System Monitor**: Real-time stats SoC, RAM, CPU/GPU frequency, temperature, uptime
- **App Profile**: Per-app optimization with automatic profiles CPU, refresh rate, and system tweaks per application

## Bug Reports

**Only bug reports with full logs will be accepted.**

For root/permission issues, include your root manager version and Android version.<br>
For app crashes, record and upload the logcat when the crash occurs.<br>
For tweak issues, include device model, SoC, and a screenshot of the Log tab.
